Setting up Mining System

  • Add the mod to your server.
  • Add the key within your key server folder.
  • Boot your server up and make sure it creates the "FreezeMining" folder within profile folder.
  • Once you see that folder you have done it.

Setting up Lifetimes

This handles lifetimes of specific items and objects that the mining system uses. Edit to your own liking.

  • Head to your root mod folder for the Mining System.
  • Copy files_miningsystem and paste within your mpmissions\mapname. For this example we will use mpmissions\dayzOffline.chernarusplus.
  • Head over to mpmissions/mapname/cfgeconomycore.xml and after the closing </defaults> tag, add the following:
<ce folder="files_miningsystem">
    <file name="miningsystem_types.xml" type="types" />
</ce>

FMS_Settings.json

0 = False / Disables | 1 = True / Enables

  • EnableLogs – Enables or disables logging
  • KeepLogDays – Number of days to retain logs
  • EnableVanillaRockMining – Toggle vanilla rock mining
  • EnableGemCuttingMiniGame – Toggle gem mini-game
  • EnableFurnace – Toggle furnace crafting
  • EnableGrindingStone – Toggle grinding station
  • EnableAnvil – Toggle anvil crafting
  • PropaneCanisterUsage – How long dose propane canister last
  • GrindingWheelUsage – How long dose the wheel last
  • AnvilHammerDamage – How long dose the Hammer last

Example FMS_Settings.json configuration:

{
    "EnableLogs": 1,
    "EnableDebug": 0,
    "KeepLogDays": 7,
    "EnableVanillaRockMining": 0,
    "EnableGemCuttingMiniGame": 0,
    "EnableFurnace": 1,
    "EnableGrindingStone": 1,
    "EnableAnvil": 1,
    "PropaneCanisterUsage": 1.0,
    "GrindingWheelUsage": 1.0,
    "AnvilHammerDamage": 0.5
}

FMS_Probabilities.json

The probability system operates out of 100 per mining circle. To guarantee a drop, set "probability": 100.

Using copper as an example:

  • probability: 70 → 70% chance on each mining circle
  • probability: 30 → 30% chance on each mining circle

Here’s what a config for mining copper might look like:

{
  "actionText": "Mine for Copper Ore",
  "max_mine": 6,
  "enabledTypes": [
    "FreezeNode_copper"
  ],
  "itemsProbabilities": [
    { "item": "Freezestone_copper", "probability": 70 },
    { "item": "Stone", "probability": 30 }
  ]
}

FMS_SpawnManager.json

0 = False / Disables | 1 = True / Enables

  • Enabled – Enable or disable this node spawn group
  • name – Unique name for the group of nodes
  • Node – Class name of the mining node
  • Spawn_min – Minimum number of nodes to spawn
  • Spawn_max – Maximum number of nodes to spawn
  • pos – Position (and optional rotation) for spawn
  • respawn_timer – Seconds before nodes respawn
  • randomize_position – Always keep this as 1

Example:

{
  "actionText": "North Copper Ore",
  "max_mine": 6,
  "enabledTypes": [
    "FreezeNode_copper"
  ],
  "itemsProbabilities": [
    { "item": "Freezestone_copper", "probability": 70 },
    { "item": "Stone", "probability": 30 }
  ]
},
{
  "actionText": "South Copper Ore",
  "max_mine": 4,
  "enabledTypes": [
    "FreezeNode_copper"
  ],
  "itemsProbabilities": [
    { "item": "Freezestone_copper", "probability": 80 },
    { "item": "Stone", "probability": 10 }
  ]
}
